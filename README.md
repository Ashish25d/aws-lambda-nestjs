## Description

Demo AWS lambda service code with nestjs and serverless framework.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start:dev

# deployment mode
$ npm run deploy
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
